# Samio

A simple script to work with AWS SAM in develop.

## Requirements

- AWS SAM CLI
- Docker
- fswatch

## Install

- Clone
  `git clone https://github.com/CeballosAndres/samio.git ~/.samio`
- Give permision
  `chmod +x -R ~/.samio/scripts`
- Add to path
  `export PATH="${HOME}/.samio/scripts/bin:${PATH}"`

## Use

`samio lambda_name`
